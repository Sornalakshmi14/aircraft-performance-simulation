# Aircraft Performance Simulation Using MATLAB

## Overview

This project simulates fundamental aircraft aerodynamic and performance characteristics using MATLAB.

The simulation calculates aircraft lift, drag, thrust required, power required, stall speed, maximum speed, maximum lift-to-drag ratio, and minimum drag speed.

## Features

* Lift vs Velocity Analysis
* Drag vs Velocity Analysis
* Lift-to-Drag Ratio Analysis
* Stall Speed Estimation
* Maximum Speed Estimation
* Thrust Required Calculation
* Power Required Calculation
* Drag Polar Visualization
* Flight Envelope Visualization
* Minimum Drag Speed Calculation
* Maximum L/D Condition Analysis

## Software Used

* MATLAB

## Key Equations

Lift:

L = 0.5 × ρ × V² × S × CL

Drag:

D = 0.5 × ρ × V² × S × CD

## Outputs

The program generates:

* Lift vs Velocity plots
* Drag vs Velocity plots
* L/D Ratio plots
* Drag Polar
* Flight Envelope
* Power Required curves
* Thrust Required curves
